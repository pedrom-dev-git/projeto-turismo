# R. Amaral — Projeto de Turismo

Website do projeto turístico R. Amaral, desenvolvido a partir do design no Figma.

## Design

- **Figma Prototype:** [Abrir no Figma](https://www.figma.com/proto/pxCxtAsVGsQj5MxlghwW75/R-amaral?node-id=45-4946)
- **Resolução:** 1440px (Desktop)
- **Documentação de Assets:** [ASSETS.md](./ASSETS.md)

## Estrutura

```
projeto-turismo/
├── assets/          → Imagens, fontes, ícones, vídeos
├── design/          → Exports do Figma e referências
├── docs/            → Documentação
├── public/          → Arquivos públicos
└── src/             → Código-fonte
```

## Como começar

1. Clone o repositório
2. Exporte os assets do Figma conforme [ASSETS.md](./ASSETS.md)
3. Organize na estrutura de pastas indicada
